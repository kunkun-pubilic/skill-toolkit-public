# Code of Conduct

This repository is a public learning and tooling package for Agent Skills. Keep discussions practical, respectful, and focused on improving the toolkit.

## Expected behavior

- Be clear and specific when reporting problems or proposing changes.
- Assume contributors have different levels of experience with Codex, Claude Code, and Agent Skills.
- Critique ideas, docs, prompts, and scripts without attacking people.
- Keep examples free of private data, credentials, customer material, and unpublished production assets.

## Unacceptable behavior

- Harassment, insults, threats, or discriminatory language.
- Posting secrets, private registry data, private runtime paths, customer content, or unrelated personal information.
- Repeated off-topic promotion or low-effort spam.
- Misrepresenting internal-only material as public project assets.

## Enforcement

Maintainers may edit, hide, or remove comments and contributions that violate this code of conduct. Repeated or severe violations may lead to being blocked from the repository.

If you see a problem, open a GitHub issue with a short description. Do not paste secrets or private data into the issue body; follow [SECURITY.md](SECURITY.md) for sensitive reports.
