# Official Basis

This public Skill is a Kun-owned overlay workflow for Agent Skill adoption.

It does not bundle a private upstream registry. During public standalone use,
the proof surface is:

- a pinned public Git source or local upstream Skill directory;
- an overlay manifest;
- generated `references/official-basis.md` and runtime references in the target Skill;
- local `skills-lock.json`;
- `quick_validate.py` and `audit_skill.py` results.

Do not claim organization-level registry promotion unless a real registry update
has happened in the user's environment.
