# Official Basis Reference

This file defines the generated `references/official-basis.md` artifact that `skill-creator` writes into every new contract-backed skill.

## Required Contents

- Generation summary: skill name, source strategy, host support, proof surface.
- Contracts used: contract IDs and local contract file paths.
- Upstream locks: upstream ID, URL, commit or docs snapshot metadata.
- Applied rules: rule ID, source contract, and rule text.

## Purpose

The generated artifact is the proof that a new skill was created from local contracts and locked official upstream sources rather than from an ungrounded template.
