# Anti-AI Voice Gates

Run these gates before calling a publish copy pack ready.

## Human Voice

Pass when the copy sounds like Kun explaining his own work:

- direct and practical;
- personal when appropriate: "我这次..." / "我发现..." / "我把...";
- no empty enthusiasm;
- no fake authority;
- no generic creator slogans.

Fail examples:

- "欢迎点赞关注, 获取更多精彩内容"
- "这期视频将带你全面深入了解..."
- "干货满满, 不容错过"
- "普通人也能轻松掌握"

## Resource Truth

Every resource promise must map to a real path, URL, or clearly marked planned resource.

- Real link exists: write the CTA.
- Link unknown: mark `needs-user`, leave placeholder, do not invent.
- Resource not ready: say it is not ready or omit it.

## Unified Positioning

All platform titles and descriptions must trace back to one `core_claim`.

Fail when:

- one platform sells a tutorial while another sells a personal story;
- one platform promises files that another omits without reason;
- titles optimize for traffic but no longer match the video.

## Publish Boundary

This skill may prepare copy and manifest fields. It must not upload, schedule, publish, or operate account UI unless a later publisher skill receives explicit authorization.
