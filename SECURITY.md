# Security Policy

This public repository intentionally excludes:

- API keys, tokens, private keys, and account credentials;
- private runtime locks and local cache paths;
- raw footage and unpublished production material;
- private organization registry data;
- customer or client project material.

The repository is a public standalone extraction of Skill lifecycle assets. It
does not include Kun's private registry, private project history, local runtime
cache, or Feishu/Lark workspace material.

## Reporting private data or security problems

If you find private data, credentials, or a security problem in this repository:

1. Open a GitHub issue with the affected file path and a short description.
2. Do not paste secrets, tokens, keys, customer data, or private registry data into the issue body.
3. If the report requires sensitive details, state that sensitive details are available privately and wait for a maintainer response.

## Supported scope

This public pack only supports the files in this repository. Kun's private registry, local runtime cache, private organization repositories, Feishu/Lark workspace material, and unpublished video production assets are out of scope for public support.
